1. ¿Por qué un lenguaje de programación sólo puede utilizarse en algunos sistemas operativos y en otros no?
R: Porque la compatibilidad de un lenguaje de programación con un sistema operativo depende de multiples factores, como la arquitectura, las bibliotecas y APIs del sistema 
   operativo. También depende si se implemente un compilador o intérprete para traducir el código fuente. 

2. ¿Qué tipo de máquina virtual es virtualBox?
R: Es una máquina virtual de tipo 2 ó hipervisor de nivel de software

3. ¿Si tengo más de una máquina virtual instalada, y una se rompe, ¿esto afecta a las demás? ¿por qué?
R: Si una máquina virtual se rompe no afecta a las demás porque cada máquina virtual se ejecuta de manera independiente y están aisladas entre sí.
